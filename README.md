# Sonos Controller for Elgato Stream Deck

This plugin provides comprehensive control over your Sonos speakers directly from your Elgato Stream Deck, including the Stream Deck+, Stream Deck MK.2, and Stream Deck Mini.

## Actions

The plugin offers a variety of actions to manage playback, volume, and favorites.

### 1. Toggle Play/Pause

This versatile action allows you to control the play state of a Sonos speaker. It's not just a button; it's a dynamic display.

*   **Play/Pause:** Press the key to toggle between playing and pausing the music.
*   **Cover Art:** The key can display the album art of the currently playing track.
*   **Track Ticker:** Shows the current track title and artist as a smooth, scrolling animation on the key.
*   **Device Name:** Optionally display the name of the Sonos speaker.
*   **Dynamic Icons:** The icon automatically switches between a "play" and "pause" symbol based on the speaker's state.

### 2. Playback Control

A multi-function action to manage track skipping and playback modes.

*   **Next/Previous:** Skip to the next track or return to the previous one.
*   **Shuffle:** Toggle the shuffle mode for the current queue. The icon changes to indicate whether shuffle is active.
*   **Repeat:** Cycle through repeat modes (Repeat All, Repeat One, Off). The icon updates to show the current repeat state.

### 3. Play Favorite

Instantly play any of your saved Sonos Favorites.

*   **Select Favorite:** Choose from a list of your saved playlists, artists, albums, or radio stations.
*   **Display Title:** The key can show the name of the favorite, scrolling if it's too long.
*   **Album Art:** Displays the cover art associated with the favorite.

### 4. Volume Control (Key)

A dedicated action for volume adjustments using standard Stream Deck keys.

*   **Commands:**
    *   **Volume Up:** Increase the volume.
    *   **Volume Down:** Decrease the volume.
    *   **Mute:** Toggle mute on and off.
    *   **Set Preset Volume:** Instantly set the volume to a specific level.
*   **Short Press / Long Press:** A short press on "Volume Up" or "Volume Down" adjusts the volume incrementally. A long press can be configured to jump to a predefined volume level.
*   **Visual Feedback:** The key can display the current volume percentage. The mute button displays a fader icon that reflects the current volume and mute status.

### 5. Volume Control (Dial for Stream Deck+)

Leverage the power of dials on the Stream Deck+ for fine-grained volume control.

*   **Rotate:** Turn the dial to smoothly increase or decrease the volume.
*   **Press:** Press the dial to toggle mute.
*   **Touch:** Touch the LCD screen to instantly set a predefined volume level.
*   **LCD Feedback:** The dial's screen shows the speaker's name, the current volume percentage (or "Muted"), and a dynamic icon that changes with the volume level.

## Common Features

*   **Automatic Device Discovery:** All actions automatically find the Sonos speakers on your network, making setup quick and easy.
*   **Real-Time State Updates:** Icons and displays update instantly to reflect the real-time status of your speakers.
*   **Efficient:** A smart connection manager ensures that the plugin communicates with your Sonos devices efficiently without spamming your network.

## Installation

1.  Download the plugin from the Elgato Marketplace.
2.  Open the Stream Deck software.
3.  Double-click the downloaded plugin file to install it.
4.  Drag any of the Sonos actions onto your Stream Deck keys.
5.  Configure the action by selecting your Sonos speaker from the dropdown list and customizing the settings.
